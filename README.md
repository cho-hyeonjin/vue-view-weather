# 뷰뷰웨더🌦️

### Vue.js 3를 이용한 날씨 애플리케이션🌦️

과거에 사용했던 Vue.js 문법을 상기하는 차원에서 진행한 🧸토이 프로젝트입니다.

Options API와 Composition API, Vuex를 사용했고 CSS는 Scss로 작성했습니다.

API는 OpenWeather API와 kakao Maps API를 사용했습니다.

## 🧑🏻‍💻 로컬에서 프로젝트 실행하기

1. `git clone https://github.com/cho-hyeonjin/vue-view-weather.git` 또는 `Download Zip` 으로 프로젝트 설치

2. `cd vue-view-weather` 경로로 진입

3. `npm install` 또는 `yarn` ( `yarn install` )

4. `yarn serve`
