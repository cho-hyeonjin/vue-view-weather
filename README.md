## 🧑🏻‍💻 로컬에서 프로젝트 실행하기

1. `git clone https://github.com/cho-hyeonjin/vue-view-weather.git` 또는 `Download Zip` 으로 프로젝트 설치

2. `cd vue-view-weather 경로` 로 프로젝트 디렉토리로 진입

3. `npm install` 또는 `yarn` ( `yarn install` )

4. `yarn serve`
